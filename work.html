face-scan

A face detection application that utilizes an API from Clarifai which uses a machine learning algorithm that detects faces from pictures uploaded by registered users. Users can register with an email and password to use the API. Users can then sign in and the app will remember the previous state of the user with the entry count. Built using React for the front-end. NodeJS and ExpressJS for the back-end. PostgreSQL for the database.

nytReact

This is a React.js app that allows users to search for New York Times articles. After performing a search, the user will see article results including the name of the article, the date and time of publishing, and a button linking to the article source. Finally, users are able to add and remove articles from a saved articles section. The app uses Node/Express for the server and routing, MongoDB/Mongoose for the database and models, Handlebars for the layout and views, & Cheerio/Request for scraping the data from www.npr.org

Jeopardy

A miniature version of the TV game show Jeopardy! This app utilizes Javascript, Jservice API, GIPHY API and Firebase which holds the leader scores.

trainTimeFirebase

Firebase Application to accept input from user to create a fictional train list and calculate the next train minutes away. Technologies used were Google Fonts, Javascript, and Firebase.
